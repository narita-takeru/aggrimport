# Description

aggrimport is aggregate import reference from go files.

# Installation

```
go get github.com/narita-takeru/aggrimport
```

# Example

```
aggrimport /Users/user/go/src/github.com/hogehoge/src
```

#### results for standard outout.

```
import  reference count
fmt     200
os      102
time    64
strings 8
```
